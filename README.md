# マイ秘書アプリ

メモ・タスク・予定・カレンダーを管理できる、自分専用の簡易秘書アプリです。
Node.js標準モジュールのみで作られているため、**npm install不要**で動作します。

## 起動方法

1. Node.js(推奨: v18以上)がインストールされていることを確認してください。
2. このフォルダ(`secretary-app`)に移動します。
   ```
   cd secretary-app
   ```
3. サーバーを起動します。
   ```
   node server.js
   ```
4. ブラウザで以下のURLを開きます。
   ```
   http://localhost:3000
   ```

サーバーを停止するには、ターミナルで `Ctrl + C` を押してください。

## 機能

画面は「ホーム」「タスク」「メモ」の3つのタブに分かれています。

- **ホーム**
  - メモ・タスク・未完了タスクの件数バッジ
  - 予定の追加(日付+内容)
  - 今日の予定 / 今日締め切りのタスク
  - 未完了タスク一覧(締め切りが近い順、期限切れは一番上)
  - 今月のカレンダー(予定・タスクの締め切りがある日に●を表示。日付をクリックするとその日の予定とタスクを確認できます)
- **タスク**
  - タスク名・締め切り日・優先度(高/中/低)・カテゴリー(仕事/勉強/趣味/買い物/その他)を指定して追加
  - カテゴリーで絞り込み表示
  - 未完了タスクは締め切り順に自動で並び替え、完了済みタスクは別の一覧で管理
- **メモ**
  - タイトルと本文を入力してメモを追加、一覧表示、削除

## データの保存場所

入力した内容は `data/notes.json`・`data/tasks.json`・`data/events.json` にそれぞれ自動保存されます。
アプリ(サーバー)を終了しても、次回起動時にデータはそのまま残っています。

## ポート番号を変更したい場合

環境変数 `PORT` を指定して起動してください。
```
PORT=4000 node server.js
```

## フォルダ構成

```
secretary-app/
├── server.js        # サーバー本体(APIとファイル配信)
├── public/
│   └── index.html   # 画面(HTML/CSS/JS)
├── data/
│   ├── notes.json   # メモの保存先(自動生成)
│   ├── tasks.json   # タスクの保存先(自動生成)
│   └── events.json  # 予定の保存先(自動生成)
└── README.md
```
